# 🚚 Parcel Management System

## Overview

The Parcel Management System is a RESTful Web API built using **ASP.NET Core** and **C#**. It is being developed incrementally following Agile sprint methodology to simulate a real-world enterprise software project.

The project demonstrates modern backend development practices including REST APIs, Entity Framework Core, SQL Server integration, authentication, and clean architecture.

---

## Technology Stack

- ASP.NET Core Web API
- C#
- .NET 8
- Entity Framework Core
- SQL Server
- Swagger / OpenAPI
- Git & GitHub
- Angular (Planned)
